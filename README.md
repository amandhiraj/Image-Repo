# ShopifyHereWeGo :) 
# Steps to run
1.) Run <code> npm install </code> on both Frontend and backend
2.) Run <code> npm start </code> in both frontend and backend.

# What I used
1.) GraphQL
2.) Node.js
3.) MongoDB
4.) React.js
5.) S3 Aws services
6.) Express.js
